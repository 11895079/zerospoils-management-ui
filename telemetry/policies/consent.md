# Telemetry Consent Policy

## MVP (M1-M3): Opt-In
- Default: Telemetry disabled
- Users must explicitly enable in Settings
- Prompt on first launch: "Help improve ZeroSpoils by sharing usage data"

## Pro Tier & Household Sync (M5+): Opt-Out
- Default: Telemetry enabled (after household setup)
- Users can disable in Settings
- Privacy policy and data use prominently disclosed

## Regional Compliance
- **Canada (Primary):** PIPEDA compliance
  - Granular consent per feature (inventory tracking, expiry insights, waste analytics)
  - Annual consent refresh required
  - Export/delete data on request (30 days)
- **GDPR (EU):** If expanding
  - Strict consent required before any collection
  - Right to be forgotten; delete all data including backups

## Data Usage
- No sale to third parties
- Analytics only: improve product, measure retention, optimize features
- No linking to ads or advertising networks
- On-device learning (future): waste prediction models trained locally

## Transparency
- Public privacy policy at zerospoils.example/privacy
- In-app policy link in Settings
- Yearly report: "Your Data" screen showing what we collected, how used
