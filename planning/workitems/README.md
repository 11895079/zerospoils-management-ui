# Workitems

All planned work should live as one file per workitem in this folder.

## Naming

Use sequential identifiers:

- `WI-0001-short-title.md`
- `WI-0002-short-title.md`

## Required Fields

- Status: `todo` | `in-progress` | `blocked` | `done`
- Owner
- Milestone
- Dependencies
- Acceptance Criteria
- Definition of Done

Use [TEMPLATE.md](./TEMPLATE.md) for new items.
